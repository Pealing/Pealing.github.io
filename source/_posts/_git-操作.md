* git clone 克隆仓库到本地
* git init 本地创建仓库
* git status 查看当前状态
* git log 查看commit的历史
* git add 
* git rm --cached [filename] 清除add中某文件
* git commit
* git push
* git branch -a 查看分支状态
* git branch [branchname] 添加一个分支到本地
* git push origin [branchname] 添加该分支到远程
* git branch -d [branchname] 删除一个本地分支
* git branch -d -r origin [branchname]在本地删除一个远程分支，但远程端依旧存在该分支
    - git branch origin [branchname] 进行这步后，完全删除远程分支
* git checkout [branchname] 将该分支作为当前分支
